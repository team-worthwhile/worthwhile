Array

Arrays are a indexed collection of elements of a given base type, which can be either Integer or Boolean. Note that because of this limitation there exists no such things like arrays of arrays.
Declaration of an array works just like primitive variable declaration. All elements to which no value is assigned on declaration, get the default value of the arrays base type. For an Integer array the default value is '0' and for an Boolean array it is 'false'. Note that arrays in worthwhile are unlimited in size and thus all elements to which are no values are assigned, have the default value.

An array element can be referenced with the specified name of the array and an non-negative Integer index.

There exists an array literal which can be used to assign values to an array. The values of an array literal can be of type Integer or Boolean but must be uniform. The indices in an array literal are continuous and start at 0.


Syntax: (Array declaration)
Integer[] <name>
Boolean[] <name>

Syntax: (Array usage)
<name>[<index>]

Syntax: (Array literal)
{(<value> (, <value>)*)?}

Example:
Boolean[] d := {true, false, true}
Integer i := 0

while(i < 3) {
	d[i] := false
	i := i + 1
}
