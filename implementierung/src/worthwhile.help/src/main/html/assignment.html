Assignment

Assignments are used to alter the value of primitive variables, single array elements or whole arrays. The new value of the variable is the value of the given expression. Note that the types of both the variable and the expression must be the same. It is also possible to alter a whole array by assigning an array literal to it.

Syntax
<name> := <expression | array literal>


Example:
Integer a
Integer b
Boolean c
Integer[] d := {21, 2}
Boolean[2] e

a := 5
b := a + a
c := true
d := {42, 4}
e[0] := d[1]
e[1] := d[0]
