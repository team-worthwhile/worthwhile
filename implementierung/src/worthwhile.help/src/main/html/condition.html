Conditions

To alter the control flow of the program, there are existing conditional statements. These statements are used to execute a certain block of statements if the given boolean expression is true. There may be an alternative block which will be executed only if the given expression is false.

Syntax:
if(<boolean expression>) {
	<if-block>
}

if(<boolean expression>) {
	<if-block>
} else {
	<else-block>
}
Example:
Integer a := 1
Integer b := 5

if( a > b) {
	a := b
} else {
	b := a
}
