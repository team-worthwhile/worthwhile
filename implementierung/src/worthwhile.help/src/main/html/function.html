Functions

In a Worthwhile program you can declare and call functions. To declare functions the 'function' keyword is used. Function declarations must be at the beginning of the code. The return type of either Boolean or Integer must be specified. Also a function is identified with a specified unique name. Optional Parameters are allowed in the declaration which can be used in the function block. You have to specify a function block that may hold statements which are executed in their own scope when this function is called. Thus it is not possible to manipulate variables of the main block or other functions. It is also not allowed to call a function before it is declared. This prevents recursive use of functions. A function is called in the main block or other functions (after the declaration) by its unique name and by passing parameters with the correct type to it. Also a function can have multiple pre- and postconditions (_requires and _ensures) which have to be placed after the parameters and before the function block. In a function block every execution flow must have a return statement. In particular this means the function block itself must have at least one return statement or every if-else must have a return statement in both, the if block and the else block.

Syntax:
function <type> <name>(<type name> <,type name>*)
<precondition | postcondition>*
{
	<statements>
}


Example:
Function declaration:
function Integer pow(Integer base, Integer exponent)
_requires(exponent ≥ 0)
{
	Integer result := base

	if(exponent = 0) {
		return 1
	}

	while( exponent > 1) {
		result := result • base
		exponent := exponent - 1
	}

	return base
}

function Boolean isEqualToSquare(Integer a, Integer b) {
	Integer c := pow(b, 2)

	if(a = c) {
		return true
	} else {
		return false
	}
}

Boolean var := isEqualToSquare(25, 5)
