Functions

In a Worthwhile program you can declare and call functions. Function declarations must be at the beginning of the code. A function has a return type of either Boolean or Integer. Each function is identified with an unique name which must start with a letter (A-Z or a-z) and followed by optional letters (A-Z or a-z), numbers (0-9), '-' or '_'. Optional Parameters are allowed in the declaration which can be used in the function block. You have to specify a function block of statements which are executed in their own context when this function is called. Thus it is not possible to manipulate variables of the main block or other functions. It is also not allowed to call a function before it is declared. This prevents recursive use of functions. A function is called in the main block or other functions (after the declaration) by its unique name and by passing the correct parameters to it. Also a function can have multiple pre- and postconditions (_requires and _ensures) which have to be placed after the parameters and before the function block.

Syntax:
function <type> <name>(<parameters>,...)
	<_requires>,... <_ensures>,...
{
	<statements>
}


Example:
Function declaration:
function Integer pow(Integer base, Integer exponent)
_requires(exponent >= 0)
{
Integer result := base

if(exponent == 0) {
	return 1
}

while( exponent > 1) {
	result := result * base
	exponent := exponent - 1
}

return base
}

function Boolean isEqualToSquare(Integer a, Integer b) {
Integer c := pow(b, 2)
	if(a = c) {
		return true
	} else {
		return false
	}
}

Boolean a := isEqualToSquare(25, 5)
