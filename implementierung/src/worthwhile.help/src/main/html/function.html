Functions

In a worthwhile program you can declare and call functions. Function declarations must be at the beginning of the code. A function has a return type of either Boolean or Integer. Each function is identified with an unique name which must start with a letter (A-Z or a-z) and followed by optional letters (A-Z or a-z), numbers (0-9), '-' or '_'. Optional Parameters are allowed in the declaration which can be used in the function block. You have to specify a function block of statements which are executed in their own context when this function is called. Thus it is not possible to manipulate variables of the main block or other functions. It is also not allowed to call a function before it is declared. This prevents recursive use of functions. A function is called in the main block or other functions (after the declaration) by its unique name and by passing the correct parameters to it.

Syntax:
function <type> <name>(<parameters>,...) {
	<statements>
}
