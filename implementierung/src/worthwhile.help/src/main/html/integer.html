Integer data type

The Integer data type is a integer value without restrictions. Following operators can be used with Integer data types:

  • The operators which have the return type Integer:

    name symbol alternative
    addtion +
    substraction -
    unary plus +
    unary minus -
    multiplication · *
    division ÷ /
    modulo %


  • The operators which have the return type Integer:

    name symbol alternative
    equality =
    inequality ≠ !=
    greater >
    greater or equal ≥ >=
    less <
    less or equal ≤ <=
