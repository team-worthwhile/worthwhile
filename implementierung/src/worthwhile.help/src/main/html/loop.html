while loop

To repeatedly execute statements while a certain expression is true there exists the while loop. A while loop has a block which is executed frequently if the given expression is true at the beginning of each cycle. In front of the block there may be invariants which tell the prover that the given expressions are true at the beginning of each loop cycle and at the end.

Syntax:
while(<expression>)
<invariant>+
{
<block>
}


Example:
Integer i := 0
while(i < 10)
_invariant i <= 10
{
	i = i + 1
}
