Worthwhile Program

A worthwhile program is a piece of code in a simple while programming language. It consists of multiple statements which form the main block of the program where the execution begins. Before the statements of the main block you are able to declare functions. There can be comments in the code which are ignored during execution.

Example:
01: //At first there may be function declarations like this:
02: function Integer pow(Integer base, Integer exponent) {
03:	Integer result := base
04:
05:	if(exponent == 0) {
06:		return 1
07:	}
08:
09:	while( exponent > 1) {
10:		result := result * base
11:		exponent := exponent - 1
12:	}
13:
14:	return base
15: }
16:
17: //After the function declarations follows the main block:
18: Integer a := 5
19: Integer b := 3
20: Integer c := 0
21:
22: c := pow(a, b)
