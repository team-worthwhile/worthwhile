Quantified Expression

A quantified expression is used within some statements like axioms, assumptions, assertions, preconditions, postconditions or invariants. These type of expression allows the user to make statements with quantors (∀ or ∃, alternative: forall or exists). The type of a quantified expression is Boolean.

Syntax:
<<∀|forall|∃|exists> <type> <name> <,<boolean expression>>?>+ : <expression>
Note: <type> can only be Integer or Boolean.

Example:
∀ Integer a ∀ Integer b,(b > 0) : a + b + 1 > a
