Statements

Statements can be used in every block, although not all statements are allowed in every block and some are essential for a block (see the respective help page).

There can be at most one statement per line. If a statement needs to span multiple lines, all lines but the last have to end in a backslash, like this: 

_ensures ∀ Integer i2, 0 < i2 ∧ i2 ≤ inputLength : \
		_return[i2] = transition[numCharacters ⋅ _return[i2-1] + input[i2-1]]

The following there is a list off all Statements:

  • Assignment
  • Variable declaration
  • Loop
  • Conditions
  • Return
  • Assertion
  • Assumption
  • Precondition
  • Postcondition
  • Invariant
  • Axiom
