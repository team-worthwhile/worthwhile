Statements

Statements can be used in every block. Although not all statements are allowed in every block and some are essential for a block (see the respective help page). Every statement has to be in his own line, that means: One line, one statement.
In the following there is a list off all Statements:

  • Assignment
  • Variable declaration
  • Loop
  • Conditions
  • Return
  • Assertion
  • Assumption
  • Precondition
  • Postcondition
  • Invariant
  • Axiom
