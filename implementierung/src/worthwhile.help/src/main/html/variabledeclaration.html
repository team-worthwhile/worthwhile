Variable declaration

Variable declarations are used to introduce new variables either of type Boolean, Integer or Array to the program. These variables are visible in the block in which they are declared and all sub blocks. It is also possible to directly assign a value at a variable declaration.
Within the program a variable is referenced with the given name.

Syntax:
Integer <name>
Boolean <name>
Integer[] <name>
Boolean[] <name>

Example:
Integer a
Integer b := 4
Integer c := b + 38
Boolean[] d := {1, 2, 3}
Boolean e := d[2]
