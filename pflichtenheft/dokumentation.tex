\subsection{Dokumentation zur Programmier- und Annotationssprache}%

\begin{anfliste}{FD}

\anf{010}{Online-Hilfe}
{Der Benutzer kann die Dokumentation zur Programmiersprache aus der Entwicklungsumgebung heraus über einen Befehl aufrufen. Diese wird dann im Browser geöffnet, sofern eine Internetverbindung vorhanden ist.}
\anf{020}{Kontextsensitive Hilfe }
{Der Benutzer kann die Dokumentation zu einem gerade ausgewählten Element der Programmier- oder Annotationssprache über einen Befehl aufrufen. Diese wird dann im Browser geöffnet, sofern eine Internetverbindung vorhanden ist.}
\anf{030}{Laden von Beispielprogrammen aus der Dokumentation heraus }
{Der Benutzer kann ein in der Online-Hilfe gezeigtes Beispielprogramm direkt aus dem Hilfefenster in den Editor laden, um es dort auszuführen.}

\end{anfliste}
